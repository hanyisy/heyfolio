HTML

시멘틱 태그를 포함한 HTML5의 핵심 요소를 활용하여 구조적이고 가독성이 높은 웹 페이지를 구현할 수 있습니다.

CSS

CSS를 활용하여 반응형 웹 디자인을 구현하고, 유연한 레이아웃을 구성할 수 있습니다.

JavaScript

ES6+ 문법을 기반으로 Canvas와 Scroll Event등의 효과를 작성할 수 있으며, 다양한 API를 활용하여 동적인 웹 애플리케이션을 개발할 수 있습니다.

sass,pug

Sass와 Pug를 활용하여 스타일과 템플릿을 조금더 효율적으로 작성하고, 유지보수하기 쉬운 구조를 구성해 보았습니다.

Jquery

jQuery를 활용하여 보다 간결하고 직관적인 코드로 DOM 조작 및 이벤트 핸들링을 효율적으로 구현할 수 있습니다.

React

컴포넌트 기반 아키텍처를 이해하고 있으며, useEffect, Hooks 등의 개념을 적용하여 동적인 웹 애플리케이션을 개발할 수 있습니다.

Node.js

Express 프레임워크를 활용하여 GET, POST, DELETE, PUT 요청을 처리하고, 간단한 방명록 등의 기능을 구현할 수 있습니다.

GCP(구글 클라우드 서비스)

GCP를 활용하여 작성한 프로젝트를 클라우드 환경에 배포할 수 있습니다.

MySQL

관계형 데이터베이스 관리 시스템(RDBMS)_MySQL에서 SELECT(JOIN 포함), INSERT, UPDATE, DELETE 등의 SQL 문을 활용하여 데이터베이스를 설계하고 최적화할 수 있으며, 효율적인 데이터 모델링과 쿼리 성능 개선을 수행할 수 있습니다.

Git & GitHub

Git을 사용하여 코드 이력 관리를 체계적으로 수행하고, GitHub를 활용한 협업 및 버전 관리를 효과적으로 진행할 수 있습니다.

Figma

피그마를 활용해 협업을 진행한 적이 있으며, 디자인 작업을 위해 사용할 수 있습니다.

Adobe_Photoshp / Illustrator / After Effect / XD

Adobe사의 프로그램들을 활용할수 있습니다.
